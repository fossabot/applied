Contributing
------------

> **Note:**  Linux is the only actively supported OS for development

**Requirements:**
- Git
- Node.js ([nvm](https://github.com/creationix/nvm) is suggested)

**Installation:**

```bash
git clone git@github.com/omahajs/applied.git
cd applied
#Install dependencies
npm install
#Verify applied installed correctly by running tests
npm test
```
**Workflow Tasks:**

- `npm test`
> run test suite
- `grunt karma:covering`
> watch task for running tests
- `grunt eslint`
> lint code
- `grunt eslinting`
> watch task for code linting
